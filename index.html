Something about me

Passion is the only thing that can drive a man of 49 to stand up every morning at 02h00 every morning to learn a new skill. I started in December 2017 with learning static web development as a hobby but by August 2018 it was much more than that.

Now it is December 2018 and my skills have grown from HTML and CSS to a point where I can call myself an intermediate web developer

All my training that I have done and still doing is from free training that I can get of the WEB as I cannot afford to study full time, or pay for distant training. I have to do all my training before and after work.

The training certificates I treasure most is most probably the ones I received from FreeCodeCamp

They include: "Front End Development", "Responsive Web Design" and "JavaScript Algorithms and Data structures"

My skills now include

HTML (using Emmet), SCSS (Corporate style setup), CSS (Grid, Grid area, Flex box), Bootstrap, JavaScript (intermediate), Express (very basic), React (very basic), and MongoDB (very basic). Very basic meaning - I have only done the theory training and must still do the practical training. I have done other introductory training like PHP, MySQL and phone app development.
